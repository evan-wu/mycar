# How to Run:

```shell script
git clone https://github.com/evan-wu/mycar.git
cd mycar
pip3 instal pyyaml adafruit-circuitpython-servokit flask pyzmq
chmod +x bin/run.sh
bin/run.sh <config_file, e.g.: config/web_drive.yml> <time_to_stop, e.g.: 120>
```

Happy and safe driving!
